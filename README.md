# Microsoft 365 Troubleshooting

Repositório público com procedimentos técnicos, troubleshooting e soluções de contorno para cenários de suporte em Microsoft 365, Azure e PowerShell.

> Todos os cenários publicados aqui são anonimizados e não contêm dados reais de clientes, usuários, tenants, domínios, chamados ou ambientes internos.

## Objetivo

Compartilhar procedimentos práticos baseados em cenários reais de suporte técnico, incluindo:

- sintomas do problema;
- causa provável;
- validações no portal Microsoft;
- comandos úteis em PowerShell;
- solução aplicada ou workaround;
- referências da documentação oficial da Microsoft.

## Categorias

- OneDrive
- Exchange Online
- SharePoint Online
- Microsoft Teams
- Microsoft Entra ID
- Azure
- PowerShell

## Aviso

Este conteúdo tem finalidade técnica e educacional.  
Os procedimentos devem ser avaliados e adaptados conforme o ambiente de cada organização.
